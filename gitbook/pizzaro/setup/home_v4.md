# Home v4

Home v4 is built using **Homepage v4 Template** with 3 rows. The Backend editor of the page look like this

![](http://transvelo.github.io/docs/pizzaro/images/home-v4-setting.png)

## Content Section

1. Navigate to **Home v4** page and make sure **Homepage v4** is choosen under **Page Attributes > Template**.
2. Click on **Publish**.
3. After Publish, **Home v4** options will appear in the meta box area.
